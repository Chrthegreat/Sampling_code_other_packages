# Package Dingo

🔗 **Repository:** https://github.com/GeomScale/dingo

*In this folder you can find the code used for Rounding the NetLib polytopes (among others) using Dingo*

---

We expect the simplified versions as input in a specific naming scheme (check the code for more)
Then, we call upon the Dingo `PolytopeSampler.round_polytope` method to round them. using one of the methods provided.
Currently only 2 of the methods work, john ellipsoid and min ellipsoid.
