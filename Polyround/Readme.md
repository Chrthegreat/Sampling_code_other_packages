# Package Polyround

🔗 **Repository:** https://gitlab.com/csb.ethz/PolyRound

*In this folder you can find the code used for Simplifying the NetLib polytopes*

---

The whole idea is this: 

- There is a folder with the files extracted from MatLab. 
- We read these files, simplify them and output the result to a different file. 
- Files are expected to be is a specific form, for example afiro_A.csv and afiro_b.csv.
